## bootstrapicons

> **Warning**
This is _not_ an official package from Bootstrap.

## Usage

```tex
%%% Example file   
    \documentclass{article}
    
        \usepackage{bootstrapicons}

    \begin{document}

        \bicon{14}

        \bicon[scale=1.2,angle=90]{29}

\end{document}
```
